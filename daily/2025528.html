今天做了什么

  • 针对设计灵感库做了小规模用户调研，访谈了台湾与北美共3位设计师，虽未使用付费竞品，但均表示对真实界面截图有需求，对 AI 灵感工具仍存观望；准备制作英文问卷，拓展至更多海外平台调研
  • 搭建设计灵感库的试验田网站，完成首页初版，接下来换英文素材将用于调研展示与付费意愿验证
  • 优化 YouTube 分析网站的分析逻辑和展示细节，增加封面图与标题等分析字段，并接入 Plausible 与 Clarity 做行为数据记录，已邀请更多朋友试用
  • 与知行午餐交流后受到启发，结合包子老师课程内容与实际需求（AI制作素材→PPT演示→录屏→剪辑），探索知识类博主视频制作工具的缺口，输出了一份关于「AI驱动的口播 + 轻动效一站式录制工具」的研究报告

  • 使用 Stitch 对视频产品的概念进行初步 UI 草图探索，但因产品方向尚不明确，输出参考度较低
  • 用上小排老师课程里讲到的 GitHub Desktop App，现在单独网页（如创业日志）上传效率显著提升，以前习惯在终端里命令行操作，现在点击两下就能完成上传，感慨自己错过了多年高效工具

有什么思考/收获

  • 国外设计师对"真实产品界面"有明确需求，但对 AI 灵感搜索工具仍持怀疑态度，产品价值需通过具体使用场景打动，还是需要可视化界面做直观演示比较好
  • YouTube 工具逐步走入实用测试阶段，行为数据收集能帮助判断真实使用频率与用户路径
    💡 洞察： 看视频也是件很好玩的事情
  • 视频内容创作者在"非剪辑型"工具上存在明显空白，"AI辅助+低学习成本"的内容生产工具具备切入机会，但可能其中技术方案会比较复杂；也去还可以尝试在其中切入更小的细分场景

碰到的卡点
